Exploration:

    We explored multiple features and settled on five simple features that, when combined and with set thresholds, would allow the code to calculate the most probable group an image could belong to within nine distinct groups.

    The five features are:

    1. Amount of blue pixels, red pixels, and contrast
    2. Brightness, whites, and blacks
    3. Saturation of red, green, and blue channels
    4. Extract amount of white and black pixels and calculate contrast
    5. Detecting images with a lot of blue, as if they were taken at sea

    When the code analyzes a provided image, all five values will be calculated, and depending on its composition, it will be determined how closely it fits within the averages of each group to determine which one it is most likely to belong to.
